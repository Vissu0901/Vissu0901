Library Management System

A library is a collection of materials, books or media that are accessible for use and not just for display purposes. A library provides physical (hard copies) or digital access (soft copies) materials, and may be a physical location or a virtual space, or both.

Additional Information

Students

Student can create account or login into application using Home Navigation. Student can borrow only 3 books once and need to get approval from any Librarian. Book need to return with in 15 days, otherwise Rs.10 fine collected for one day from return date. If Student fails to return Book within 15days of 5 consective failure leads to defaulter(account will be blocked for next 1.5 months).

Librarians

Librarian can login into application or new account using Job Code provided by Management for registration. Librarian need to check his/her pending tasks and make necessary action on Student Request. Please check Task details before you approve/reject. Please try to complete pending tasks asap before EOD.

Management

Management team can login into application, have access to add, update and remove book details. Management can check Student and Librarian details.
